# Costa Rica AI Partner

Landing page for **Costa Rica AI Partner** — a consultancy helping small and medium businesses in Costa Rica leverage AI, data engineering, and cloud infrastructure to modernize their operations.

## Live Site

👉 **[costaricaaipartner.com](https://kvn2099.github.io/costa-rica-ai-partner/)**

## About

Costa Rica AI Partner helps SMBs:

- **Gain visibility** — dashboards, clean data, and automated reports.
- **Automate workflows** — CRM/billing integrations, AI assistants, and process orchestration.
- **Scale infrastructure** — cloud setup, data pipelines, backups, and ongoing support.

The site is a single-page landing built with vanilla HTML and CSS (no frameworks, no build step). It is fully responsive, supports Spanish, and is designed for fast load times.
